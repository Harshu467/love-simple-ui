## Customizable Buttons

Buttons are html elements that can be used to trigger actions. They can be customized to fit your needs.
We provides a large variety of styles and sizes for the button component including outlined buttons, multiple colors, sizes, buttons with icons, and more.

<Steps>

### Installation
To use the buttons, you need to install the package first and complete the setup.
Setup instructions can be found [here](https://love-simple-ui.vercel.app/Installation).

### Import
Import the buttons in your project.

```js
import { ButtonComponent } from 'love-simple-ui'
```

### Enjoy using the buttons
You can now use the buttons in your project and customize them to fit your needs. ðŸŽ‰
</Steps>
