### Switch

A customizable switch component with color, checked state, and icon support.

<Steps>

### Installation

To use the switch, you need to install the package first and complete the setup.
Setup instructions can be found [here](https://love-simple-ui.vercel.app/Installation).

### Import

Import the switch in your project.

```js
import { Switch } from "love-simple-ui";
```

### Enjoy using the switch

You can now use the switch in your project and customize them to fit your needs. ðŸŽ‰
